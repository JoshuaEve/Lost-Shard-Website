# Commands

All LostShard commands are split into two categories: **player commands** available to everyone, and **admin commands** restricted to staff with the `lostshard.admin` permission.

- [Player Commands →](/commands/player-commands)
- [Admin Commands →](/commands/admin-commands)

## Quick Reference

### Character & Stats

| Command | Description |
|---------|-------------|
| `/stats` | View your gold, murder count, mana, and stamina |
| `/skills` | Open the skill menu |
| `/title` | Open title selection GUI |
| `/dps` | Toggle damage-per-second display |

### Social & Chat

| Command | Description |
|---------|-------------|
| `/global [msg]` / `/g [msg]` | Switch to or send global chat |
| `/party [msg]` / `/p [msg]` | Party commands and chat |
| `/clan [msg]` / `/c [msg]` | Clan commands and chat |
| `/whisper <player> <msg>` | Send a private message |
| `/shout <msg>` | Broadcast a colored message |

### Magic

| Command | Description |
|---------|-------------|
| `/cast <spell\|1-9>` | Cast a spell by name or bound slot |
| `/bind <slot> <spell>` | Bind a spell to a number key |
| `/scrolls` | View and manage spell scrolls |
| `/runebook` | Open runebook for teleportation |
| `/meditate` | Toggle meditation (faster mana regen) |

### Economy & Storage

| Command | Description |
|---------|-------------|
| `/bank` | Open your 27-slot bank |

### Land & Groups

| Command | Description |
|---------|-------------|
| `/plot create <name>` | Create a new plot |
| `/plot info` | Open plot management GUI |
| `/claim` | Start capturing a control point |
| `/party invite <players>` | Invite players to your party |
| `/clan create <name>` | Create a new clan |

### Skills & Combat

| Command | Description |
|---------|-------------|
| `/blacksmithy` | Open blacksmithy GUI |
| `/fishing` | Fishing commands |
| `/survivalism` | Survivalism skill menu |
| `/pets` | Open pet management menu |
| `/friendlyfire [on\|off]` / `/ff` | Toggle friendly fire |
| `/spawn` | Teleport to spawn |
