# Survivalism

**Survivalism** is a utility skill focused on wilderness survival and camp management. Players with high Survivalism can create persistent camps that serve as field bases — useful for long expeditions far from spawn.

## Leveling Survivalism

Survivalism gains experience through camp-related activities. Use `/survivalism` to open the Survivalism menu and access camp creation options.

## Camp System

Camps are temporary field bases that you can establish in the wilderness:

- **Create a camp** via the Survivalism menu
- Camps persist between server restarts
- The Camp Manager handles setup and cleanup automatically

## Rebirth: Pathfinder

Once Survivalism reaches **1000**, you can rebirth the skill. Rebirthing resets Survivalism to 0 and permanently unlocks the **Pathfinder** passive.

## Commands

```
/survivalism
```

Opens the Survivalism skill menu.

## Tips

- Camps are most useful on long expeditions where returning to spawn would waste time.
- Pair Survivalism with a [Runebook](/magic/runebook) for maximum mobility — mark your camp location with the Mark spell and recall back at any time.
