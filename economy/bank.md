# Bank

Every player has a personal **bank** — a **27-slot (3-row)** virtual chest that can be accessed from anywhere on the server. Bank contents persist across sessions and server restarts.

## Opening Your Bank

```
/bank
```

## Features

| Feature | Details |
|---------|---------|
| Storage size | 27 slots (3 rows) |
| Access | From anywhere — no special location needed |
| Persistence | Survives logouts and restarts |
| Per-player | Entirely private |

## Admin Access

Admins can open any player's bank using:

```
/getbank <player>
```

This is intended for moderation and support purposes.

## Tips

- Use your bank to store rare materials and valuables you don't want to risk losing on death.
- Spell scrolls can also be managed via `/scrolls` — but physical item backups belong in the bank.
- The bank does not hold gold directly — gold is a character stat managed separately.
